# PharmaHeaven 💊
### Wholesale Pharmacy Management System

A comprehensive, full-stack management platform designed for pharmaceutical wholesalers to manage inventory, sales, suppliers, and business analytics through a modern, responsive dashboard.

---

## 📽️ Project Overview
PharmaHeaven is an all-in-one solution for medical distributors to digitize their operations. It provides real-time tracking of medications, bulk sales processing for healthcare providers, and deep insights into supply chain health.

### 🌟 Key Features
- **📊 Interactive Dashboard:** Real-time metrics for total products, orders, suppliers, and revenue.
- **📦 Intelligent Inventory:** Track stock availability across categories with dynamic "Low Stock" status alerts.
- **💰 Sales & Purchase Management:** Record detailed wholesale transactions with automated total calculations.
- **🤝 Supplier/Category Management:** Database to organize and manage wholesale partners and product brands.
- **🔍 Advanced Search:** Instant filtering for thousands of medicinal products.
- **🔐 Secure Authentication:** Role-based access for system security.
- **📱 Responsive UI:** Optimized for performance and clarity across all devices.

## 🛠️ Technology Stack
- **Frontend:** HTML5, CSS3 (Modern Flexbox & Grid), JavaScript (ES6+, Fetch API).
- **Backend:** PHP (API and server-side logic).
- **Database:** MySQL.
- **Tools:** FontAwesome, Google Fonts (Outfit), XAMPP for local hosting.

## 🚀 Getting Started
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Furqan15380/Pharma-Heaven.git
   ```
2. **Setup Database:**
   - Import `database.sql` into your PHPMyAdmin / MySQL Server.
3. **Configuration:**
   - Ensure your database credentials match in `config.php`.
4. **Run:**
   - Host the directory in your XAMPP/Apache `htdocs` folder and navigate to `index.html`.
